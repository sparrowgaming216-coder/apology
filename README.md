# 💌 Sorry Letter for Anjali

A heartfelt, interactive apology letter created with HTML, CSS, and JavaScript.

## 📝 About

This is a beautiful, animated sorry letter expressing genuine remorse for accidentally call bombing. The letter includes:

- 💖 Animated, heartfelt apology message
- ✨ Beautiful animations and transitions
- 🌹 Decorative floating flowers
- 💝 Interactive buttons for forgiveness
- 📱 Option to send custom messages
- 🎉 Confetti animation on forgiveness

## 🚀 How to Use

1. Open `index.html` in your web browser
2. Read the heartfelt apology letter
3. Click on the interactive buttons to respond

## 📦 Features

- **Responsive Design**: Works on desktop, tablet, and mobile
- **Smooth Animations**: Fade-in effects for text and floating decorations
- **Interactive Buttons**:
  - 💝 "I Forgive You" - Shows appreciation message with confetti
  - 📱 "Send a Message" - Write a custom response
- **Beautiful UI**: Gradient backgrounds and smooth transitions

## 💻 Technologies

- HTML5
- CSS3 (with animations)
- JavaScript (for interactivity)
